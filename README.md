# Sistema de Órdenes de Reparación

Aplicación web para crear órdenes de reparación de maquinaria conectada con Smartsheet.

## 🚀 Deploy Rápido (1 clic)

### Opción 1: Heroku (Recomendado)
Haz clic en el botón:
[![Deploy to Heroku](https://www.herokucdn.com/deploy/button.svg)](https://heroku.com/deploy?template=https://github.com/tu-usuario/sistema-ordenes)

### Opción 2: Railway
1. Ve a https://railway.app
2. Haz clic en "New Project"
3. Selecciona "Deploy from GitHub"
4. Autoriza tu GitHub
5. Selecciona este repositorio
6. ¡Listo! Railway desplegará automáticamente

### Opción 3: Render
1. Ve a https://render.com
2. Haz clic en "New+"
3. Selecciona "Web Service"
4. Conecta tu GitHub
5. ¡Listo!

## 📋 Características

- Formulario responsivo para crear órdenes
- Se conecta con Smartsheet automáticamente
- Funciona en celular y PC
- Notificaciones automáticas al técnico
- Interfaz moderna y fácil de usar

## 🛠️ Desarrollo Local

```bash
npm install
npm start
```

Abre http://localhost:3000
